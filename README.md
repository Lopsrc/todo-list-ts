# Todo List
The todo list has a simple functional. Authentication, Roles, Docker, PrismaORM.

## Docker

Build and run the Docker container:
```
docker-compose up
```

## Local

Install dependencies:
```
npm install
```

Run migrations.
```
./migrate.sh
```

### Run the application:
```
npm run start:dev
```

### Run the tests:
```
npm run test
```
